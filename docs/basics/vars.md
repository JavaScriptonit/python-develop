# Переменные:
1. Создание переменных:
    ```
    >>> count = 40
    >>> PI = 3.14
    >>> is_fun = True
    >>> greet = 'Hello'
    ```
2. Адресс переменных:
    ```
    >>> id(PI)
    4395459888
    >>> id(count)
    4393009752
    ```
3. Типы переменных:
    ```
    >>> type(count)
    <class 'int'>
    >>> type(is_fun)
    <class 'bool'>
    ```
